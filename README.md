# ZZTI / 智障钛

一个公开可访问的中文短剧脑回路测试站。

在线访问：

- https://emajjsky.github.io/zzti/

## 这是什么

`ZZTI` 源于对 `MBTI / SBTI / XBTI` 这类测试玩法的恶搞重构，但方向不是正经心理学，而是：

- 测你在中文短剧情境里的反应模式
- 算你的“脑残指数”
- 把结果映射到短剧题材和角色位

它不是“你像不像某个角色”，而是先测：

- 上头度
- 恋爱脑度
- 装逼度
- 复读度
- 刻薄度
- 降智度
- 做局度
- 掉马妄想度

再输出：

- 核心人格结果
- 脑残指数
- 适配短剧赛道
- 高频角色位

## 当前站点能力

- 500 题结构化题库
- 16 个核心人格
- 2 个隐藏人格
- 当前公开站固定为 `发疯卷`：50 题
- 首页已切成更强的暴躁嘲讽风，并接入 `wan2.7` 生成的站点抽象图
- 纯静态前端，可直接部署到 GitHub Pages

## 当前仓库内容

这个仓库当前是 **站点发布仓库**，主要放静态页面：

- `index.html`
- `styles.css`
- `app.js`
- `data/question-bank.v1.json`
- `.nojekyll`

也就是说，现在这个仓库的职责是：

- 承载公开测试页面
- 承载前端题库数据
- 通过 GitHub Pages 对外提供访问

下一阶段准备把它从“静态测试站”往“可运行技能”推进，重点会做：
